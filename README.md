## Tours and Travels Company

A React Frontend project that displays beautiful website of a Tours and Travels Company.

_Check out the live demo of the project_ [here](https://longroads-travels-react.netlify.app) ‚≠ê

## Installation and Setup Instructions

Clone down this repository. You will need `node` and `npm` installed globally on your machine.

#### Installation:

`npm install`

To Run Test Suite:

`npm test`

To Start Server:

`npm start`
